Groups

Groups are perhaps the most exciting feature of Emplify. Groups allow you to create micro-communities around departments, geography, events or some other common factor. It's kind of like WhatsApp, but for work.

Groups need to be created by Administrators, and you can only see Groups, once an administrator has added you, but once you're in you can share content that is only accessible by people in that group.
