Getting Started

Administrators can login to Emplify at go.emplifyapp.com/admin/. Once you're logged in you can carry out any of the following:

  • Create, edit, delete News post
  • View and delete users Community posts
  • View, edit and delete categories (used in News)
  • Post questions for feedback and view Results
  • Create groups
  • Create users and assigning groups
